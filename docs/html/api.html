Exceptions

exception powfacpy.PFInterfaceError

There should always be a base class (that inherits from ‘Exception’) for all custom errors/exceptions.

exception powfacpy.PFAttributeError(obj, msg_raised, pf_base_interface)

Attempt to access an invalid attribute of a PF object.

exception powfacpy.PFAttributeTypeError(obj, attr, msg_raised, pf_base_interface)

Attempt to set an invalid type for the attribute of a PF object.

exception powfacpy.PFPathError(existing_path, non_existing_child)

Attempt to access invalid path in PF database.

exception powfacpy.PFNonExistingObjectError(folder, obj, condition=False, include_subfolders=False)

Attempt to access PF object that does not exist.
